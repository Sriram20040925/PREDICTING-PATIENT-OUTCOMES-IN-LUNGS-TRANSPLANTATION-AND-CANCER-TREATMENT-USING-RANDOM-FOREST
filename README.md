# PREDICTING PATIENT OUTCOMES IN LUNG TRANSPLANTATION AND CANCER TREATMENT USING RANDOM FOREST

## 📌 Project Description

This project focuses on predicting patient outcomes in lung transplantation and cancer treatment using the Random Forest machine learning algorithm. It analyzes clinical and medical data to identify patterns affecting survival rates and treatment success, helping healthcare professionals make informed decisions.

---

## 🎯 Objectives

* Predict patient survival and treatment outcomes
* Improve accuracy using Random Forest algorithm
* Assist doctors in decision-making
* Reduce risk and improve patient care

---

## 🧠 Methodology

The system uses the Random Forest algorithm, an ensemble learning technique that builds multiple decision trees and combines their outputs to improve prediction accuracy and reduce overfitting.

---

## 📊 Features

* Data preprocessing and cleaning
* Feature selection
* Model training using Random Forest
* Outcome prediction (Survival / Risk level)
* Performance evaluation
